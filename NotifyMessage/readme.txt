		You can copy NotifyMessage files to any folder.

		You need .NET Framework 4.5.

	Files:
		NotifyMessage.exe
		readme.txt

	Usage:
		NotifyMessage.exe [opt] "Your message"

	Option:
		/INFO(default) | /WARN | /ERR
